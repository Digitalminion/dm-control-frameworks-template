# Templates Folder

## Overview

This folder contains a **complete suite of standardized templates** for generating consistent documentation across the NIST documentation system. Templates provide AI agents with structured formats and clear instructions for creating both organizational context documentation and comprehensive NIST control documentation.

## What You'll Find Here

### Core Context Templates
- **`README.tpl.md`** - Template for standardizing README file structure across folders
- **`organization-profile.tpl.md`** - Template for organizational profile information
- **`technology-stack.tpl.md`** - Template for technology infrastructure documentation
- **`risk-profile.tpl.md`** - Template for risk assessment and compliance requirements

### ✅ **Complete Control Template Suite**
- **`controls/` folder** - Complete suite of NIST control documentation templates
  - **`control-overview.tpl.md`** - Main control documentation and description
  - **`implementation-guide.tpl.md`** - Step-by-step implementation instructions
  - **`assessment-framework.tpl.md`** - Evaluation criteria and testing procedures
  - **`compliance-tracking.tpl.md`** - Audit requirements and evidence collection
  - **`risk-assessment.tpl.md`** - Control-specific risk considerations and mitigation
  - **`dependencies.tpl.md`** - Related controls and system dependencies

### Template Purpose

Templates serve as:
- **Standardized formats** for consistent documentation structure
- **AI agent guidance** with clear instructions and examples
- **Quality assurance** through predefined validation rules
- **Efficiency tools** for rapid documentation generation
- **Comprehensive coverage** for all aspects of NIST control documentation

## Using Templates

### For AI Agents

#### Context Documentation
1. **Identify the appropriate template** for your documentation needs
2. **Follow the template structure** and instructions exactly
3. **Use the provided examples** as a starting point
4. **Apply the validation rules** to ensure quality
5. **Generate content** in the appropriate context/ subfolder

#### NIST Control Documentation
1. **Go to `methods/templates/controls/` folder** for complete control template suite
2. **Read the controls/README.md** for comprehensive guidance
3. **Select appropriate template** based on documentation type needed
4. **Follow AI instructions** in each template header
5. **Apply organizational context** from context/ folder
6. **Replace all placeholders** with actual control-specific content
7. **Generate comprehensive documentation** for each NIST control

Templates provide:
